# Pydaria

Pydaria API

## Requirements

- python3 == 3.8.10
- pip == 22.3.1
- virtualenv == 20.17.1

## Project setup (development env) ðŸ› 

Create a new venv folder

```
virtualenv venv
```

Activate venv

```
source venv/bin/activate
```

Install project dependencies

```
pip install -r requirements.txt
```

Create a new .env file

```
cp .env.example .env
```

Run Flask in development mode

```
flask run
```
